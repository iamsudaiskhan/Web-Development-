My Projects

DodgeFall – Python Survival Game

DodgeFall is a fast-paced survival game developed using Python and Pygame, where players dodge falling objects that increase in speed and frequency as the game progresses. I implemented real-time collision detection and a dynamic scoring system that adapts to the player's performance. The project helped refine my understanding of game loops, object physics, and event handling in a 2D environment. Through this, I demonstrated the ability to build an interactive and progressively challenging game experience from scratch.

Pong Battle – Two-Player Competitive Game

Pong Battle is a Python-based remake of the classic arcade game, enhanced with additional logic and a scoring system for competitive play. I implemented collision detection for both paddles and walls, built win/loss condition handling, and designed a responsive UI for real-time multiplayer gameplay. This project strengthened my skills in structuring game states, managing user input, and applying physics-based logic to maintain smooth game mechanics.
